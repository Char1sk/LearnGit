# 廖雪峰Git笔记

## 创建版本库

创建本地仓库

> git init

把工作区文件添加到版本库

> git add _file_  
> git commit -m "_msg_"

## 时光机穿梭

掌握仓库当前状态

> git status

比较未add的修改内容

> git diff _file_

### 版本回退

查看现有版本

> git log
> git log --pretty=oneline

回退曾经版本

> git reset --hard HEAD^^
> git reset --hard HEAD~2

跳转某个版本

> git reset --hard _commit-id_

查看记录改动命令及版本号

> git reflog
